Grid Structure

To create an organized space, we used twelve column grid that was helpful to work with while creating a page layout.

Colour Swatches

White: primary colour

#FFFFFF should be the most widely used across all screens and components

Shade of Grey: secondary colour

#808080 should be used to indicate a related action or information

Shade of Orange: accent colour

#FD6A4F should be used for interactive elements, such as links, buttons, and text cursors
